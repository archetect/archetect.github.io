# archetect.github.io

This is the source [mdbook](https://rust-lang.github.io/mdBook/) project for the [Archetect Book](https://archetect.github.io/).

## Book Development

This book is built with [mdbook](https://rust-lang.github.io/mdBook/). To install `mdbook`,  assuming the [Rust Toolchain](https://rustup.rs/)
is already installed:

```shell
cargo install mdbook
```

The book can be served locally with:

```shell
mdbook serve -o
```
