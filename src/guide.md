# Guide

This guide takes you through the basics, and then presents new capabilities and features within each section.  The 
completed work of each section is captured as branches in the [archetect-tutorial](https://github.com/archetect/archetect-tutorial) Git 
repo, which you can check out and review, or even render with Archetect.

If you'd like to use the companion tutorial, first switch into a directory where you keep projects, and then clone 
tutorial's Git repo:

```shell
    git clone https://github.com/archetect/archetect-tutorial.git
```

You visit each section by using git to list them out and then check out the branch that interests you.

```shell
    git branch
    git checkout 01_basics
```
