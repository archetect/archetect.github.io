# Reference

Normative reference documentation for Archetect v3. For task-oriented guides, see the [User Guide](../user-guide/) and [Authoring Archetypes](../authoring-archetypes/).

| Section | Description |
|---|---|
| [CLI](./cli/) | Every `archetect` command, flag, and environment variable. |
| [Lua API](./lua-api/) | The complete Lua scripting API available to `archetype.lua`. |
| [Templating (ATL)](./templating/) | Archetect Template Language syntax and filter reference. |
| [Archetype Manifest](./archetype-manifest) | Full `archetype.yaml` schema: metadata, `requires`, `templating`, `catalog`, and `interface`. |
| [Catalog Manifest](./catalog-manifest) | `CatalogEntry` schema: leaf, group, and server entries, plus the `library` and `show` flags. |
| [Configuration Files](./configuration) | `archetect.yaml` tool configuration: file locations, merge order, all sections, environment variables. |
| [Answer Files](./answer-files) | Answer file formats, `-a key=value` typed syntax, and answer precedence. |
| [Archetype Layout](./archetype-layout) | The normative directory and file layout of an archetype on disk. |
